Greetings! Here is the website I made for the Final Project in the Advanced Styling with Responsive Design course in Coursera. The layout of the page like the div of the page will change based on the size of the screen. So it will be responsive to any changes that you make to the screen. This is done through media queries that are written in the styles.css file.
